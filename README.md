## ✏️ 기본 기능

### 🚨 패킷 필터링
1. **Pyshark를 통한 패킷 덤프**
   - Pyshark를 통하여 패킷을 덤프한 다음 웹 서버와 탐지/차단 시스템으로 데이터를 반환합니다!

2. **위협 탐지**
   - 문자열 기반으로 위협을 탐지/차단합니다!
   - 이미 공개된 정보로 IP를 탐지/차단합니다!
   - 사용자의 규칙으로 위협을 탐지/차단합니다!

### 📈 모니터링 시스템

1. **데이터를 시각화**
   - 웹 서버를 통하여 패킷,트래픽을 시각화하여 이용자가 더욱 편하게 데이터를 확인을 할 수 있습니다!

2. **편의 기능**
   - 사용자는 웹을 통하여 규칙,이벤트,계정을 관리할 수 있습니다!
   - 또한 시스템에서 발생한 모든 이벤트, 로그들을 볼 수 있습니다!
